# jasmine-sample
This is an example of using jasmine for TDD and istanbul for code coverage.

To use this example:

Clone the repo:

```bash
git clone https://github.com/rbultman/jasmine-sample
```

Set up Jasmine:

```bash
cd jasmine-sample/
npm init
```

Run the tests:

```bash
npm test
```

To get code coverage:  istanbul cover jasmine
